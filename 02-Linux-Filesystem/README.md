# 📂 Module 02: Linux Filesystem

> **"Everything in Linux begins with understanding the filesystem."**

Welcome to the Linux Filesystem module.

In this module, you'll learn how Linux organizes files and directories, why every path starts from `/`, and how the filesystem forms the foundation of the entire operating system.

By the end of this module, you'll be comfortable navigating the Linux filesystem and understand where applications, users, and system files live.

---

# 🎯 Learning Objectives

After completing this module, you will be able to:

* Explain what a filesystem is.
* Understand the Linux Filesystem Hierarchy.
* Identify the purpose of important Linux directories.
* Differentiate between the Root Directory and the Root User.
* Understand absolute and relative paths.
* Explain why Linux follows the philosophy of **"Everything is a File."**
* Navigate the filesystem with confidence.
